Fonte

Todo componente eletrônico precisa de uma certa quantidade de energia para poder funcionar, isso inclui as peças do computador. A fonte de alimentação é um componente fundamental para a máquina, nela que irá ocorrer o processo de transformação de energia alternada para energia contínua, para que todo o sistema funcione sem problemas.
